## Root Cause Analysis

### Determination: Intentional test exception (not a production bug)

The `TrafficSimulationController` at `src/Controller/TrafficSimulationController.php` is a **deliberate load-testing endpoint** designed to exercise the Auto-RCA pipeline under controlled stress. Its behavior is:

1. Accepts only requests bearing a valid `X-Sim-Token` or `key` query parameter.
2. Tracks in-flight requests via an atomic APCu counter.
3. Burns CPU for ~20 seconds (configured via `TRAFFIC_SIM_DURATION` env var, default 20).
4. **Throws a `RuntimeException`** if the number of concurrent in-flight requests exceeds `TRAFFIC_SIM_MAX_CONCURRENCY` (default: `1`).

In this incident, 2 concurrent `curl` requests hit the endpoint `/fr/admin/auto-rca/simulate-traffic-exception` with threshold 1, causing the second request to throw as designed. The `/fr` locale prefix is standard Symfony i18n routing.
