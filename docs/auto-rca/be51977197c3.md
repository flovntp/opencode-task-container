### Verdict: Intentional exception ‚Äî no code bug

This exception was thrown **deliberately** by the `TrafficSimulationController`, which is a load-testing endpoint designed to exercise the Auto-RCA pipeline. The controller:

1. Tracks concurrent in-flight requests via an atomic APCu counter (`traffic_sim.inflight`)
2. Compares the count against a threshold (default: 1, from `TRAFFIC_SIM_MAX_CONCURRENCY`)
3. If the number of concurrent requests exceeds the threshold, it throws a `RuntimeException` to simulate load shedding

In this incident, 2 concurrent requests arrived nearly simultaneously while the threshold was 1. The second request incremented the counter to 2, immediately failed the guard at `TrafficSimulationController::assertWithinThreshold()` (line 115, called from `process()` at line 68), and threw the exception with `elapsed 0s`.
